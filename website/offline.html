❓ Frequently Asked Questions

Do I need internet to use AbyssBox Offline? ▼
No! AbyssBox Offline works completely without internet. All features are available locally. The only time you might need internet is to download the MP3 codec on first use, but WAV export works completely offline.
What's the difference between Complete and Minimal editions? ▼
The Complete Edition includes all sound banks (Kirby, Mario, Wario, etc.) and all features. The Minimal Edition has just the essential instruments and drums. Complete is ~150 MB, Minimal is ~40 MB. Both include all effects and synthesis capabilities.
How do I save and share my songs? ▼
All song data is stored in the URL. When you make changes, the URL updates automatically. Simply copy the URL from your browser address bar and share it with others. They can paste the URL to recreate your exact song. You can also shorten long URLs using the File menu.
What's the difference between WAV and MP3 export? ▼
WAV is lossless and highest quality - works completely offline. MP3 is compressed and smaller - requires internet on first use to download the codec. Both work great, choose WAV for archival and MP3 for sharing online.
What is microtonal tuning? ▼
Microtonal tuning allows you to use tuning systems other than standard 12-note equal temperament. AbyssBox supports 1-1200 EDO (Equal Divisions of the Octave), including presets like 19 EDO (neutral thirds), 24 EDO (quarter-tones), 53 EDO (Pythagorean), and more.
What if I have performance issues? ▼
Try closing other browser tabs, reducing the number of active patterns, or using a modern browser (Chrome/Firefox recommended). If using file:// directly, try running a local web server instead. This usually resolves any performance issues.
Is my data collected or tracked? ▼
No! AbyssBox Offline is completely privacy-first. No data is collected, no tracking, no telemetry. Everything runs locally on your computer. Your songs are stored only in the URL - nowhere else.
Can I use this commercially? ▼
Yes! AbyssBox is released under the MIT License. You can use it commercially, modify it, and redistribute it. Songs you create belong to you - we claim no ownership. See the credits file for third-party licenses.
Where can I get help or report bugs? ▼
Check the included manual.html for detailed help. For bugs and feature requests, visit the GitHub repository at github.com/coripolanco79-source/abyssboxwithmicrotuning. The community is friendly and helpful!
